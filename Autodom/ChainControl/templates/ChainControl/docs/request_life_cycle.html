Варианты изменения статуса заявки

{% endblock%} {% block body %}

Заявки имеют несколько вариантов перехода между статусами.


На согласовании -> Согласована -> Выполнена

Это самый просто вариант перехода заявки между сатусами. Инициатор создает заявку (статус: На согласовании), заявка одобрена согласующими (статус: Согласована), исполнитель выполняет заявку (статус: Выполнена).



На согласовании -> На доработке -> На согласовании

В этом варианте согласующий отправляет заявку на доработку. Инициатору необходимо отредактировать информацию в заявке и сохранить изменения. После сохранения заявка автоматически вернется в статус На согласовании.



На согласовании/Согласована -> Отменена

Заявка может быть отменена в любой момент инициатором или согласующим в момент рассмотрения. Исключением являются заявки со статусом Выполнена, они не редактируются.
