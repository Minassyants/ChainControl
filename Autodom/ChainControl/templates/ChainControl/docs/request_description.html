Описание структуры заявки

{% endblock%} {% block body %}

Заявка содержит информацию о согласуемом счете на оплату, контрагенте и ожидаемом сроке исполнения платежа.

  1. Данные заявки

    1. 1. Вид заявки – определяет участников согласования данной заявки.
    2. 2. Срок выполнения – срок выполнения данной заявки.
    3. 3. Тип оплаты – тип оплаты по данной заявке.

  2. Данные о контрагенте

    1. 4. Контрагент – контрагент, указанный в счете на оплату.
    2. 5. Договор – выбираете договор контрагента.
    3. 6. Номер счета – номер счета контрагента.
    4. 7. Сумма – сумма счета на оплату.
    5. 8. Валюта – валюта счета на оплату.

  3. Данные счета на оплату

    1. 9. Номер счета – номер счета на оплату.
    2. 10. Дата счета – дата счета на оплату.
    3. 11. Дата оказания услуг – дата акта выполненных работ, связанного с счетом на оплату.
    4. 12. Содержание – содержание счета на оплату.
  1. 13. Комментарий – комментарий к заявке.
  2. 14. В заявке присутствует возможность прикрепления 3 файлов.
