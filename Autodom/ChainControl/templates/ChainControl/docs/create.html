Инструкция по созданию заявки

  1. На главном экране нажать кнопку "Создать заявку".
  2. Внести информацию о заявке. Поля "Комментарий", "Содержание" и "Прикрепленные файлы" могут быть пустыми. Если поле "Валюта" оставить пустым, значение будет взято из договора контрагента.
  3. Сохранить и отправить заявку на согласование нажатием кнопки "Отправить заявку".


Инструкция по изменению заявки

  1. На главном экране открыть заявку из списка "Мои заявки".
  2. Внести необходимые изменения.
  3. Сохранить заявку нажатием кнопки "Сохранить". При изменении заявки, статус автоматически устанавливается На согласовании!


Инструкция по отмене заявки

  1. На главном экране открыть заявку из списка "Мои заявки".
  2. Отменить заявку нажатием кнопки "Отменить".
